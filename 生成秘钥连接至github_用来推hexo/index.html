生成秘钥连接至github————用来推hexo

发表于 2023-06-18 阅读次数：

生成秘钥

邮箱自然是我的git注册邮箱

如果修改passphrade密码

直接enter即密码为空，这是秘钥的密码

复制秘钥到github


打开/root/.ssh/id_ed25519.pub
复制到这里

https://github.com/settings/ssh/new

PS官方已不推荐用rsa加密

测试


这里输入yes

成功！
